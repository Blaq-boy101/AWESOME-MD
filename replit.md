# AWESOME-MD WhatsApp Bot

## Overview
WhatsApp Multi Device Bot project.

## Project Structure
- `index.js` - Main entry point
- `main.js` - Main bot logic
- `config.js` - Configuration settings
- `lockpackage.js` - Lock package utilities
- `session` - Session storage file
- `package.json` - Node.js dependencies

## Running the Project
Run with: `node index.js`
